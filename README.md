# Todo List Website

A simple website for managing a todo list with server-backed persistence.

## Files
- `index.html` — page structure and app layout
- `styles.css` — visual design and responsive styling
- `script.js` — interactive todo list behavior
- `server.js` — Node.js backend API and static file host
- `todos.json` — persisted task storage
- `package.json` — dependencies and run scripts

## Run locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Open `http://localhost:3000` in your browser.
